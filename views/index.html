[IN DEVELOPMENT] Image Search Abstraction Layer App by Alexander Kallaway

One of Free Code Camp Backend Challenges

Search for images

With this app, you can search for images and specify the page of results you want to view, and the app will send you a JSON with results back. The example below is looking for "funny animals" and displays page 9 of results.

https://pic-search.glitch.me/api/imagesearch/funny%20animals?page=9

View recent searches

You can also view recent searches by going to this page:

https://pic-search.glitch.me/api/latest/imagesearch
